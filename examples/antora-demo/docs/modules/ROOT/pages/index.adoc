= asciidoctor-kroki-embedded Antora Demo
:kroki-default-format: svg

This Antora page is generated with the local `asciidoctor-kroki-embedded` extension registered through `asciidoc.extensions` in the playbook.
The extension accepts Kroki-compatible diagram blocks and emits safe, inert HTML targets that an Antora UI or host application can hydrate with local renderers.

== How this demo is wired

[source,yaml]
----
asciidoc:
  extensions:
    - ./antora-kroki-embedded-extension.mjs
----

[source,js]
----
import { createAntoraExtension } from '../../src/antora.js'

const extension = createAntoraExtension({
  defaultFormat: 'svg',
  diagramNames: ['mermaid', 'plantuml', 'graphviz', 'vegalite'],
})

export function register(registry, context) {
  extension.register(registry, context)
}
----
